The Django Lost and Found System is a web-based application developed using Django.
This system allows users to report lost items, post found items, and search for matching records.
It helps in managing lost and found items efficiently within an organization, campus, or community.

🚀 Features

User Registration and Login

Post Lost Items

Post Found Items

View Item Details

Search and Filter Items

Admin Panel for Managing Posts

Secure Authentication System

🛠 Technologies Used

Python

Django

HTML

CSS

Bootstrap

SQLite Database
